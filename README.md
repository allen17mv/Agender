# Agender
Age - Gender detection using opencv and CNN


# Dataset
Age - https://www.dropbox.com/s/xfb20y596869vbb/age_net.caffemodel?dl=0

Gender - https://www.dropbox.com/s/iyv483wz7ztr9gh/gender_net.caffemodel?dl=0

python AgeGender.py
